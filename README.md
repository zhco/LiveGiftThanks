# 直播礼物自动感谢助手

## 功能
监控直播间屏幕上的礼物提示（如"XXX 送出 嘉年华"），自动在聊天区发送"感谢XXX送来的嘉年华！"文字。

## 使用步骤

1. 用 Android Studio 打开本项目，编译安装到手机
2. 打开 App，点击按钮跳转到系统"无障碍设置"
3. 找到"直播礼物感谢"，开启服务
4. 进入直播间即可自动工作

## 注意事项
- 首次使用建议用小号测试，确认发送内容无误
- 不同直播平台 UI 差异大，可能需要微调 GiftThanksService 中的正则和节点查找逻辑
- 无障碍服务可能被系统省电策略杀掉，建议添加到白名单
- 3秒防抖：同一人同一礼物短时间只感谢一次，避免刷屏

## 项目结构
```
app/src/main/java/com/example/livegiftthanks/
├── MainActivity.java          # 入口界面
└── GiftThanksService.java     # 核心服务（无障碍监听 + 自动发送）
```
